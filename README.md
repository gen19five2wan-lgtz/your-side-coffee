# YOUR SIDE COFFEE

三鷹駅徒歩2分のスペシャルティコーヒー専門店の公式ウェブサイト

## 特徴

- 🌍 60種類以上の厳選コーヒー豆
- ⚡ オンデマンド焙煎（注文後に焙煎）
- 📍 JR三鷹駅南口から徒歩2分
- 🐕 ペット可（テラス席）
- 📶 Wi-Fi完備

## サイトの特徴

- レスポンシブデザイン（PC/タブレット/スマートフォン対応）
- 豆のフィルタリング機能（産地別）
- Pinterest風クラフト美学
- SEO最適化済み

## 技術スタック

- HTML5
- CSS3（カスタムデザイン）
- Vanilla JavaScript
- Google Fonts (Noto Sans JP / Noto Serif JP)

## ローカルでの起動方法

```bash
# HTTPサーバーを起動
npx http-server -p 8000

# または Python
python -m http.server 8000
```

ブラウザで `http://localhost:8000` を開く
